# Программа на С++ для ведения личного учёта денежных средств

```
Главное меню:
1. Посмотреть баланс
   ├── Посмотреть баланс
   ├── Посмотреть по дате
   └── Посмотреть по классификации
       └────Доходы
       |    ├── Зарплата
       |    ├── Подработка
       |    └── Удача      
       └────Расходы
            ├── Налоги и жилье
            ├── Продукты питания
            ├── Развлекательные расходы
            └── Непредвиденные расходы  
                
2. Посмотреть отчёт
   ├── Полный отчёт
   ├── Посмотреть по дате
   └── Посмотреть по классификации
3. Управление счётом
   ├── Добавление записи
   ├── Удаление записи
   └── Редактирование записи
```
